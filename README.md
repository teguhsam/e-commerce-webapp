# ecommerce-webapp

eCommerce Web App with Golang

# Requirements

```
brew install make
brew install mariadb
```

1. go
2. make
3. mariadb
4. bootstrap (https://getbootstrap.com)
5. Air (https://github.com/air-verse/air)
   run `air`

# To run application

```
go run ./cmd/web
```
